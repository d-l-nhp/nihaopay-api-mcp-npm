# nihaopay-api-mcp

NihaoPay API MCP server npm package.

## Install

```bash
npm install -g nihaopay-api-mcp
```

## Add to Claude Desktop

```json
{
  "mcpServers": {
    "nihaopay-docs": { "command": "nihaopay-mcp" }
  }
}
```

## Tools

- `search_docs`
- `fetch_doc`
- `get_error_code`
- `list_endpoints`
- `list_docs`

## Env

Min node version: 22
